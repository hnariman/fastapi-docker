# Sample FastAPI project with Docker

### Build
```
$ docker-compose build
```

### Start
```
$ docker-compose up
```

### Stop
```
$ docker-compose down
```

### Description
Just a sample to show how FastAPI can be Dockerized (as a boilerplate)

  FastAPI goodies:
  /docs & /redoc endpoints head to swagger and OpenAPI 3.0 documentation
